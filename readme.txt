Maze
===============
Typical maze game for SE313 Computer Graphics and 
Visual Programming by using Android SDK and jPCT library.

DEPENDENCIES:

	This project is done with Samsung Galaxy Tab with Android 2.2
	First install Eclipse and Android SDK, then open the files in Eclipse.
	
HOW TO PLAY:
	
	To control the ball, tilt screen left and right
